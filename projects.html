Project Overview

A responsive web application for task management built with vanilla JavaScript. Features include drag-and-drop functionality, local storage, task categorization, and priority management. The app works offline and syncs when back online.

The Challenge

Create an intuitive, fast task management tool that works seamlessly across devices without requiring user accounts or server infrastructure.

Built a Progressive Web App (PWA) using modern JavaScript features. Implemented local storage for data persistence and service workers for offline functionality. The drag-and-drop interface makes task management intuitive.
